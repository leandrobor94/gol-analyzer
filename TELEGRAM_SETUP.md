# Cómo crear un bot de Telegram

1. Abre Telegram y busca **@BotFather**
2. Envía: `/newbot`
3. Ponle nombre: `GolAlert` (o el que quieras)
4. Ponle username: `tugolalert_bot` (debe terminar en `_bot`)
5. Te dará un **token** como: `7234567890:AAHdqTcvCH1vGWJxfSeofSAs0K5PALDsaw`
6. Busca tu bot (el username que creaste) y envíale `/start`
7. Busca **@userinfobot**, envíale `/start` y te dará tu **ID numérico**

Guarda esos dos valores:
- Token del bot
- Tu ID numérico
